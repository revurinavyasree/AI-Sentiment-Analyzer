# 🧠 AI Sentiment Analyzer

## Internship Project

**Intern ID:** CITS1184

This project was developed as part of the internship program and demonstrates the use of Natural Language Processing (NLP) and Streamlit to analyze sentiment from user-provided text.

---

## Project Overview

AI Sentiment Analyzer is a Python-based NLP application that determines whether a piece of text expresses a Positive, Negative, or Neutral sentiment.

The application provides an interactive Streamlit dashboard, maintains analysis history, visualizes sentiment trends, and allows users to download results for further analysis.

---

## Features

- Sentiment Analysis using NLP
- Positive, Negative, and Neutral classification
- Sentiment score calculation
- Interactive Streamlit dashboard
- Analysis history tracking
- CSV export functionality
- Sidebar sentiment filtering
- KPI metrics dashboard
- Pie chart visualization
- Bar chart visualization
- User-friendly interface

## Installation

Clone the repository:

```bash
git clone https://github.com/revurinavyasree/AI-Sentiment-Analyzer.git
cd AI-Sentiment-Analyzer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python -m streamlit run app.py
```
